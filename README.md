# Recipes Website

Full-stack recipe website built with Vue.js, Node.js and MySQL.

## Features
- User registration and login
- Recipe search using Spoonacular API
- Save favorite recipes
- Recently viewed recipes
- Create personal recipes

## Tech Stack
Frontend: Vue.js  
Backend: Node.js + Express  
Database: MySQL  
External API: Spoonacular

## Running the project

### Backend
cd backend  
npm install  
npm start

### Frontend
cd frontend  
npm install  
npm run serve
